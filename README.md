# 🚲 SmartBike: Demand-Aware Rental & Management System

**COMP 202 - Software Engineering | Spring 2026**

## 📝 Proje Hakkında
SmartBike, bisiklet paylaşımını dijitalleştirirken şehir operasyonlarını veri analitiği ile optimize eden web tabanlı bir kiralama platformudur. Standart uygulamalardan farklı olarak, çevresel verileri (hava durumu, talep yoğunluğu vb.) entegre ederek kullanıcılara akıllı kiralama önerileri sunar.

## 🎯 First Iteration Notes (İterasyon 1 Hedefleri)
Bu aşamada projenin temel altyapısı kurulmuş ve aşağıdaki gereksinimler planlanmıştır:

* **Req1 - Kullanıcı Kayıt & Giriş:** Güvenli şifre yönetimi ile kullanıcı kimlik doğrulama sisteminin kurulması.
* **Req2 - Veritabanı Tasarımı:** En az 5 tablodan (Users, Bikes, Zones, Reservations, DemandData) oluşan ve Foreign Key ilişkileri tanımlanmış ilişkisel veritabanı mimarisi.
* **Req3 - Bölge ve İstasyon Yönetimi:** Kullanıcıların farklı bölgeleri (Midtown, OldTown vb.) ve bu bölgelerdeki bisiklet kapasitelerini görebilmesi.
* **Req4 - Veri Entegrasyonu:** `city_bike_demand_2025.csv` veri setinin akıllı analizler için sisteme aktarılması.
* **Req5 - Arayüz İskeleti:** Sidebar (Profil, Bisiklet Rezervasyonu, Rezervasyonlarım, Akıllı Öneriler) içeren ana sayfa yapısının oluşturulması.

## 🛠️ Teknik Altyapı (Tech Stack)
* **Backend:** Python / Django
* **Database:** MySQL (Relational DBMS)
* **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript
* **Version Control:** Git & GitHub
